# configurações

## frontend: 

> npm install

> npm start

## backend:

> npm install

> node ace generate:key

copiar o arquivo `.env.example` com o novo nome de `.env` e colocar o valor gerado acima na APP_KEY

> node ace migration:run

> node ace serve
